An attempt to make a countdown timer that will always be accurate

How this works:

  1. You'll note the {{ timerLength }} second countdown timer below. Refresh the page to reset the timer
  2. If navigating away doesn't pause the timer, in chrome you can pause the timer with the following steps:
    1. Press F12 to open developer tools
    2. Press F8 to navigate to the javascript source tab
    3. Press F8 again to pause the javascript
  3. Wait a few seconds, and note the timer is NOT counting down
  4. When you're ready to resume, press F8 again or click the resume button in the top menu
  5. Note the timer resumes where it would have left off and continues normally.
